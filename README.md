# Retail Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales data to uncover sales trends, regional performance, and product category insights.  
The goal was to demonstrate end-to-end data analytics skills including data cleaning, SQL analysis, and Power BI visualization.

---

## Tools Used
- Microsoft Excel – Data cleaning and preparation  
- SQLite – SQL analysis  
- Power BI – Interactive dashboard creation  

---

## Dataset
Retail sales dataset containing:
- Order information
- Customer details
- Product categories
- Regions
- Sales values

---

## Key Analysis Performed (SQL)
- Total sales and total orders
- Sales by region
- Sales by category
- Top products by revenue
- Monthly sales trends

---

## Power BI Dashboard Features
- Total Sales KPI  
- Total Orders KPI  
- Sales by Region (Bar Chart)  
- Sales by Category (Column Chart)  
- Monthly Sales Trend (Line Chart)  
- Interactive slicers for:
  - Region  
  - Category  
  - Segment  

---

## Key Insights
- The West region generated the highest sales overall.
- Technology products contributed the most revenue among categories.
- Sales showed clear monthly trends indicating seasonal demand.
- Consumer segment accounted for the largest share of orders.

---

## Project Structure
- `data/` – Raw dataset  
- `sql/` – SQL queries used for analysis  
- `powerbi/` – Power BI dashboard file  
- `visuals/` – Dashboard screenshots

This project demonstrates practical data analytics skills including:
